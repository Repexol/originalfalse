## 合成
这个mod有多种合成方法：分别是原初节点和召唤台。

原初节点的合成配方看json文件（不过别指望编辑它达到魔改的效果，这只是文本的一部分，配方是代码写死的）

按照要求防止后，请确保准确无误，然后拿着原初转化剂右键一下，就可以收获了，空手右键查看内部空间。

这东西的容量是无限的。~~专业垃圾桶~~
召唤台只有5个东西的合成。

召唤台本身是可以合成的。

原初节点作为矿物生成/召唤台合成。
